Problem

You have a number and you want to print it with thousands and decimal separators. For example, you want to display the number of people who have viewd page, or the percentage of people who have voted for an option in a poll.

  1.   $number = 1234.56;
  2.  
  3.   $formattedNumber = number_format($number, 2, ".", ",");
  4.  
  5.   print "$formattedNumber";

Problem

The number_format() automatically rounded the number after decimal point. To avoid this

  1.     $number = 31415.92653;
  2.  
  3.     list($int, $dec) = explode('.', $number);
  4.  
  5.     // print "\n$int\n$dec";
  6.     // print "\n".strlen($dec);
  7.  
  8.     $formattedNumber = number_format($number, strlen($dec));
  9.  
  10.     print "$formattedNumber";
