Generating random number is usefull:

  • Display random image on a page
  • Randomize the starting position of game
  • Select a random record from a database
  • Generate unique session identifier

Php has two differant random number generators, a classic function called rand() and a better function called mt_rand(). MT stands for Mersenne Twister, which is named after the Frech monk and mathematician Marin Mersenne. mt_rand() is less predictable and faster than rand().
